# 问卷数据处理工具 - 本地部署使用指南

## 快速开始

### 一键安装（推荐）

1. 双击运行 `一键安装.bat`
2. 等待安装完成（首次需要下载依赖包，约 1-3 分钟）
3. 双击 `启动服务.bat` 启动服务
4. 打开浏览器访问：http://localhost:5000

## 手动安装步骤

如果一键安装失败，可以按照以下步骤手动安装：

### 1. 安装 Python

- 访问 https://www.python.org/downloads/
- 下载 Python 3.9 或更高版本
- 安装时**必须勾选** "Add Python to PATH"

### 2. 安装依赖

打开命令行，执行以下命令：

```bash
cd 问卷数据处理工具所在的目录
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
```

### 3. 启动服务

```bash
venv\Scripts\activate
python app.py
```

## 如何在其他电脑上部署

### 方法一：打包整个文件夹（最简单）

1. 复制整个项目文件夹到 U 盘或通过网盘发送
2. 在目标电脑上运行 `一键安装.bat`
3. 完成！

### 方法二：创建独立安装包

使用 PyInstaller 打包成独立可执行文件（需要额外配置）

## 使用说明

### 启动服务

每次使用前需要启动服务：

- 双击 `启动服务.bat`
- 或在命令行执行：`venv\Scripts\activate && python app.py`

### 访问应用

服务启动后，通过以下方式访问：

1. 打开浏览器，输入：http://localhost:5000
2. 或双击 `问卷数据处理工具.url` 快捷方式

### 停止服务

在启动服务的窗口中按 `Ctrl + C`，或直接关闭窗口

## 注意事项

### 端口冲突

如果 5000 端口被占用，会提示错误。可以修改 `app.py` 最后一行的端口号：

```python
app.run(debug=True, host='0.0.0.0', port=5000)  # 改为其他端口，如 8080
```

### 防火墙设置

- 本机访问无需特殊设置
- 如果需要局域网内其他电脑访问：
  - 需要修改 `app.py` 中的 `host='0.0.0.0'`
  - 需要在防火墙中允许 Python 通过

### 网络要求

- 本地部署完全离线，无需网络连接
- 仅首次安装时需要网络下载依赖包

## 常见问题

### Q: 提示 "Python 不是内部或外部命令"？

A: Python 未正确安装或未添加到环境变量。
1. 重新安装 Python，确保勾选 "Add Python to PATH"
2. 或手动添加 Python 到系统 PATH

### Q: 启动后无法访问 http://localhost:5000？

A: 检查以下几点：
1. 确认服务窗口没有错误信息
2. 尝试其他端口（修改 app.py）
3. 检查防火墙设置

### Q: 如何在同一局域网内访问？

A: 需要两步：
1. 修改 `app.py` 最后一行为：`app.run(host='0.0.0.0', port=5000)`
2. 其他电脑访问：`http://你的电脑IP:5000`（如 http://192.168.1.100:5000）

### Q: 如何查看电脑 IP 地址？

A: 打开命令行，输入：`ipconfig`
查看 "IPv4 地址" 一项
